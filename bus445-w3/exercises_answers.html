Exercise 4.5.11: Describing Seasonal Patterns

Task: Relevel the Month and Easter factors to have chronological order and a logical base case (e.g., ‚ÄúJanuary‚Äù and ‚ÄúNon Easter‚Äù).

# Check current levels
levels(Eggs$Month)

 [7] "June"      "March"     "May"       "November"  "October"   "September"
# Relevel Month to chronological order
Eggs$Month <- factor(Eggs$Month, levels = c("January", "February", "March", "April", "May", "June", "July", "August", "September", "October", "November", "December"))

# Relevel Easter to set "Non Easter" as reference
# Note: stringsAsFactors=TRUE likely made "Easter" and "Non Easter" levels. 
Eggs$Easter <- relevel(Eggs$Easter, ref = "Non Easter")

# Refit the model with releveled factors to interpret seasonal patterns correctly
LinearEggs_Releveled <- lm(Cases ~ . - Week, data = Eggs)
