Introduction

This document analyzes the multicollinearity among variables in the dataset. Multicollinearity occurs when independent variables in a regression model are highly correlated, which can make it difficult to determine the individual effect of each variable.

Correlation Matrix (Numerical Variables)

Fix Applied: We use method = "color" instead of circles. This ensures that even very weak correlations (near 0) are visible as colored tiles, and the specific numbers are printed for clarity.

numeric_vars <- analysis_df %>% select(where(is.numeric))

# Drop columns with zero variance (constants)
is_constant <- sapply(numeric_vars, function(x) sd(x, na.rm = TRUE) == 0)
numeric_vars <- numeric_vars[, !is_constant, drop = FALSE]

if (ncol(numeric_vars) >= 2) {
  # Use Spearman to be robust to nonlinearity/skew
  cor_matrix <- suppressWarnings(cor(numeric_vars, use = "pairwise.complete.obs", method = "spearman"))
  
  # Plotting fix: method="color" handles low values better than circles
  corrplot(cor_matrix, 
           method = "color",        # Use tiles (squares)
           type = "lower",          # Show lower triangle only (cleaner)
           addCoef.col = "black",   # Print the numbers
           tl.col = "black",        # Black labels
           tl.srt = 45,             # Rotate labels
           diag = FALSE,            # Remove the self-correlation diagonal
           number.cex = 0.8,        # Adjust text size
           title = "Correlation Matrix (Spearman)", 
           mar = c(0,0,2,0))        # Adjust margins to prevent title cutoff
} else {
  print("Not enough numeric variables with variance to compute a correlation matrix.")
}

Categorical Variable Association (Cramer’s V)

Since many variables are categorical, Pearson correlation doesn’t apply. We use Cramer’s V to measure association between categorical variables (0 = no association, 1 = perfect association).

# Identify categorical variables
cat_vars <- analysis_df %>% select(where(is.factor))
cat_names <- names(cat_vars)

# Initialize matrix
n_cat <- length(cat_names)
cramers_v_matrix <- matrix(NA, nrow = n_cat, ncol = n_cat, dimnames = list(cat_names, cat_names))

# Function for Cramer's V
get_cramers_v <- function(cat1, cat2) {
  tbl <- table(cat1, cat2)
  # Chi-squared test
  chi_res <- suppressWarnings(chisq.test(tbl, correct = FALSE))
  chi2 <- chi_res$statistic
  n <- sum(tbl)
  r <- nrow(tbl)
  k <- ncol(tbl)
  # Cramer's V calculation
  sqrt(chi2 / (n * min(r - 1, k - 1)))
}

# Fill matrix
for (i in 1:n_cat) {
  for (j in 1:n_cat) {
    if (i == j) {
      cramers_v_matrix[i, j] <- 1
    } else {
      cramers_v_matrix[i, j] <- get_cramers_v(cat_vars[[i]], cat_vars[[j]])
    }
  }
}

corrplot(cramers_v_matrix, 
         method = "color", 
         type = "lower", 
         title = "Cramer's V (Categorical Association)", 
         is.corr = FALSE, 
         cl.lim = c(0, 1), 
         addCoef.col = "black",
         tl.col = "black",
         diag = FALSE,
         mar = c(0,0,2,0))

Numeric vs Categorical Association (R-squared)

To check if a categorical variable strongly predicts a numerical variable, we calculate the \(R^2\) from a simple linear regression (Numeric ~ Categorical).

num_vars <- analysis_df %>% select(where(is.numeric))
num_names <- names(num_vars)

# Initialize matrix
assoc_matrix <- matrix(NA, nrow = length(num_names), ncol = length(cat_names), 
                       dimnames = list(num_names, cat_names))

for (n in num_names) {
  for (c in cat_names) {
    # R-squared from ANOVA/Linear Model
    mod <- lm(analysis_df[[n]] ~ analysis_df[[c]])
    assoc_matrix[n, c] <- summary(mod)$r.squared
  }
}

corrplot(assoc_matrix, 
         method = "color", 
         title = "R-squared (Numeric vs Categorical)",

Variance Inflation Factor (VIF) Analysis

VIF is a measure of the amount of multicollinearity in a set of multiple regression variables.

# Fit a linear model (dummy model for VIF check)
lm_model <- lm(amount ~ class + study + gpa + greek + off_campus + distance_to_campus + gender + sprinklered + coverage, data = analysis_df)

# Check for aliased coefficients (perfect multicollinearity)
if (any(is.na(coef(lm_model)))) {
  print("Perfect multicollinearity detected (likely off_campus vs distance). Refitting without 'off_campus' for VIF calculation.")
  lm_model <- lm(amount ~ class + study + gpa + greek + distance_to_campus + gender + sprinklered + coverage, data = analysis_df)
}

# Calculate VIF
vif_values <- vif(lm_model)
print(vif_values)

Conclusion

Summary of Findings

  1. Numerical Correlations: High values indicate redundancy. Since values are low, linear correlation is minimal.
  2. Categorical Associations: High Cramer’s V values (close to 1) indicate variables carry similar information.
  3. Numeric vs Categorical: High R-squared values suggest a categorical variable predicts a numeric one.
  4. VIF Analysis: If off_campus caused aliasing, it confirms redundancy with distance_to_campus.
